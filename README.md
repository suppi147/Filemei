# Filemei

### Introduction
- Filemei is an online file upload system with time expired period.

### Ability
- upload, download file

### Technology used
- Backend
    - Java Servlet
- Frontend
    - JSP, CSS, Javascript

### Future development
- frontend
    - limit alert
- Backend
    - file timeout
    - secure download folder
