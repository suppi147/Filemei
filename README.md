# Filemei

### Introduction
- Filemei is an online file upload system with time expired period.

### Ability
- upload files, images.
- QR link for copy.
- interactive with noteziee

### Technology used
- Backend
    - Java Servlet
- Frontend
    - JSP, CSS
- Database
    - MySQL

### Future development
- frontend
    - add upload,down FE
- Backend
    - upload multifile, folder
